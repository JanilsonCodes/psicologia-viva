### Online Psychology Consultation Platform

## 📖 About

Psychology Viva is a frontend project focused on creating a modern and accessible platform for booking psychological consultations.

The project was built with a strong emphasis on user experience, responsive design, and intuitive navigation, making it easy for users to find information and schedule appointments.
